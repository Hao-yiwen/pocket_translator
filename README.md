  - 🎯 简洁优雅的原生 macOS 风格

  - 🎛️ 可自定义 API 密钥配置

## 安装说明：

- 下载 TranslatorGenerator.dmg
- 打开 DMG 文件
- 将应用拖入 Applications 文件夹
- 首次运行时右键点击应用选择"打开"

## 常见问题

如果提示"无法打开应用程序"，请尝试：

1. 右键点击应用选择"打开"
2. 在系统设置的安全性与隐私中允许打开
3. 如果仍然无法打开，请在终端中运行：

```bash
xattr -cr /Applications/pocket_translator.app
```
